# 🧁 Shah's Bakery - User Portal

![React](https://img.shields.io/badge/React-18-61DAFB.svg?logo=react)
![Vite](https://img.shields.io/badge/Vite-Enabled-646CFF.svg?logo=vite)

Welcome to the **User Portal** for Shah's Bakery. This is the customer-facing storefront where users can browse the bakery catalogue, add items to their cart, and place orders seamlessly.

## ✨ Features
- **Product Catalogue:** Browse delicious baked goods with dynamic search and filtering.
- **Shopping Cart:** Add products to the cart and view totals in real-time.
- **Checkout & Orders:** Secure payment processing and order history tracking.

## 📁 Folder Structure
The main `src/` directory is organized as follows:
```text
src/
├── app/        # Application-level configuration and global stores.
├── components/ # Reusable UI elements.
├── features/   # Domain-specific modules (e.g., Cart feature, Auth feature).
├── hooks/      # Custom React hooks.
├── lib/        # Third-party library initializations and utility wrappers.
├── routes/     # Application routing definitions.
├── styles/     # Global styling.
└── utils/      # Helper functions and pure utilities.
```

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v16 or higher)
- NPM or Yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/amankrmj09/bakery-website.git
   cd bakery-user
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```
